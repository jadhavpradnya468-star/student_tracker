
# 📊 Student Tracker App

A powerful Student Productivity Tracker built using Flask, MySQL, HTML, CSS, and JavaScript.
Track your learning, build consistency, and improve study habits with analytics and smart insights.

---

## 🚀 Features

* 📚 Subject Management (Add / Delete / Major Subjects)
* 📖 Daily Learning Logs with study hours
* 🔥 Streak System for consistency tracking
* 📊 Study Analytics (Graph-based visualization)
* 📝 Notes Section for each subject
* ✅ To-Do Task Manager (Mark & Delete tasks)
* 🎯 Goal Tracking System
* 🧠 Smart Insights (Weekly analysis)
* ⚠️ Burnout Detection System
* ⭐ Focus Score Calculation
* 🤖 AI Assistant (basic chatbot)
* ⏱️ Pomodoro Timer

---

## 🛠️ Tech Stack

* **Backend:** Flask (Python)
* **Database:** MySQL
* **Frontend:** HTML, CSS, JavaScript
* **Charts:** Chart.js

## ⚙️ Installation & Setup

1️⃣ Clone the repository
git clone https://github.com/jadhavpradnya468-star/student_tracker.git

2️⃣ Navigate to project folder
cd student_tracker

3️⃣ Install dependencies
pip install -r requirements.txt

4️⃣ Setup MySQL database

* Create database: `student_tracker`
* Import tables

5️⃣ Run the app
python app.py
